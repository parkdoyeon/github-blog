javascript - 정규식으로 문자열 replace하기

Sep 17 2019

'2019-09-17'을 '2019/09/17'으로 바꾸려면 .replace()함수만으로는 안된다. replace()는 가장 먼저 등장하는 문자열만 교체하기 때문이다. 때문에 전체 문자열에서 변경사항을 적용하려면 다음과 같이 정규식을 사용해야한다.

var item = '2019-09-17'
item.replace(/-/g, '/')

정규식 끝에 있는 소문자는 ‘전체탐색’을 의미한다. i와 함께 많이 사용되는데, 정리하면 아래와 같다.

  • i: ignore case, 대문자/소문자 구분하지 않음
  • g: global, 전체 탐색

MDN 문서에 좋은 예시들이 많이 나오는데, 아래와 같은 사용도 가능하다.

var re = /(\w+)\s(\w+)/;
var str = 'John Smith';
var newstr = str.replace(re, '$2, $1');
console.log(newstr);  // Smith, John
