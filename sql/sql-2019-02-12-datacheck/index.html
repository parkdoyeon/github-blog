데이터별 갯수 확인 쿼리

Feb 12 2019

집계 데이터중에서 특정갯수만큼의 열을 갖고있는 데이터를 확인하고싶으면 having을 통해 확인하면된다

SELECT [varchar_col]
FROM [table]
GROUP BY [varchar_col]
HAVING count(*) > 1;

group by 인자가 여러개일경우 순서에 따라서 집계된다.
