Mac OS에서 asdf을 통한 언어 버전관리 + elixir 설치하기

Feb 1 2020

asdf는 개발환경에서 언어 사용 버전을 관리할 수 있도록 하는 오픈소스이다. 솔루션마다 적용되는 언어가 다를 경우 개발환경에서 커맨드 라인을 통해 손쉽게 현재 사용되는 언어의 버전 포인터를 바꿔줄 수 있다. 설치나 사용방법이 간단하고, asdf 공식문서에도 설명이 잘 나와있지만 할때마다 elixir와 erlang 세팅을 할때 한번씩 빼먹는게 있어서 순서대로 정리해놓기로 했다. 내가 사용하는 환경은 Mac OS Catalina 이상에 패키지 매니저로 homebrew가 설치되어있다.

  1. asdf설치하기 (Mac OS 중심으로)
    Homebrew를 통해 asdf을 설치한다.

    $ brew install asdf

    Mac OS Catalina 버전에서 다음과 같이 쉘에 입력한다. 

    $ echo -e '\n. $HOME/.asdf/asdf.sh' >> ~/.zprofile

    미만 버전에서는 아래를 입력하면 된다.

    $ echo -e '\n. $HOME/.asdf/asdf.sh' >> ~/.bash_profile
$ echo -e '\n. $HOME/.asdf/completions/asdf.bash' >> ~/.bash_profile

  2. 플러그인 추가하기
    각 버전을 관리할 수 있도록 플러그인을 추가한다

    $ asdf plugin-add erlang
$ asdf plugin-add elixir

  3. erlang 설치하기
    이제 언어를 설치하면 되는데, elixir는 erlang VM환경에서 동작하므로 erlang이 설치되어 있어야 한다. 그리고 erlang을 설치를 위해서는 build tool을 설치해야 한다. Mac OS의 경우 이미 있을 확률이 높다.

    $ brew install autoconf

    확인이 되었다면 이제 erlang OTP를 다운로드하자. 나는 현재 최신버전인 22.2를 받았다. 다운로드가 완료되면 다운로드 받은 erlang언어 버전중에 사용하고싶은 버전을 global하게 지정해주면 된다. local로 지정하면 로그인한 시스템 유저만 사용할 수 있다.

    $ asdf install erlang 22.2
$ asdf global erlang 22.2

  4. elixir 설치하기
    erlang이 잘 설치되었다면 이제 엘릭서 설치를 하면 된다.

    $ asdf install elixir 1.10
$ asdf global elixir 1.10

    설치가 완료되면 아래와 같은 명령어를 통해 리스폰스가 잘 나오는지 확인해보자.

    $ elixir -v
$ iex # elixir의 인터렉티브 쉘

  5. asdf 사용 현황 보기

    $ asdf current
elixir         1.10     (set by /Users/doyeon/.tool-versions)
erlang         22.2     (set by /Users/doyeon/.tool-versions)
linux asdf
